About us

Multi-activity Wikipedia in an interactive and entertaining way is what WikiPedro brings to you.

We provide visualizing Wiki search results in a transparent format and making related articles visible immediately.You can also build your own graph libraries to Manage personal browsing preference perceptively, possibly save the results for later use and import local files to library.

WikiPedro is a web application that uses data from Wikipedia API and visualization by D3.js.
